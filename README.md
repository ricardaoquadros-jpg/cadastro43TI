# Cadastro43TI – API PHP para Aplicações Móveis

Este repositório contém uma API REST simples desenvolvida em PHP e PDO para manipulação de contatos (CRUD: Create, Read, Update e Delete).  
O projeto foi construído como parte da disciplina **Programação para Dispositivos Móveis**, servindo como backend para um aplicativo Android criado no Android Studio.

A API foi estruturada no **Visual Studio Code** e desenvolvida com o auxílio de ferramentas de IA modernas, incluindo **ChatGPT 5.1** para geração e otimização de código e **Gemini 3 Pro** para refinamento e validações complementares.

## Tecnologias utilizadas
- PHP 8+
- PDO (MySQL)
- XAMPP/LAMPP
- Estrutura MVC simples
- JSON para comunicação com Android (Volley)
- Git e GitHub para versionamento

## Finalidade
Servir como base de dados e API para práticas de comunicação entre aplicativos Android e servidores web na disciplina *Programação para Dispositivos Móveis*.

## Funcionalidades
- Inserção de contatos
- Consulta por nome ou listagem completa
- Atualização de registros
- Exclusão de contatos
- Respostas padronizadas em JSON
- Especialmente otimizada para consumo por apps Android (Volley)
